# **WEEK 10 WORKLOG**

### **Week 10 Objectives**

* Learn about container orchestration with **Kubernetes (K8s)**.
* Master core Kubernetes concepts (Cluster, Node, Pod) and how to write YAML configuration files.
* Practice deploying an application (Node.js) to a Kubernetes cluster (using Minikube).
* Learn about container monitoring tools (Prometheus, Grafana, cAdvisor).
* Learn about Kubernetes backup and disaster recovery solutions using **Velero**.
* Review documentation and prepare for the final CI/CD presentation.
