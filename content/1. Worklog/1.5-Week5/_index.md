### **Week 5 Achievements**

Week 5 focuses on summarizing knowledge learned, in-depth learning about application security and updating AI trends in software development. Achievements include:

#### 1. Summarize & write report for 4 weeks

* Collect all learned content: IAM, VPC, EC2, Auto Scaling, CloudWatch, S3, CloudFormation
* Write comprehensive report according to standard format:
  * Objectives
  * Tasks by day
  * Achievements
  * Lessons Learned
* Connect knowledge between services, understand relationships in AWS architecture

#### 2. Translate technical blog (AWS/Cloud)

* Choose content related to cloud-native architecture
* Translate blog to Vietnamese to understand terminology:
  * Load Balancer
  * Fault tolerance
  * Horizontal scaling
  * Zero Trust
* Improve reading comprehension of technical documentation

#### 3. Protect applications with AWS WAF

* Understand Layer 7 protection mechanism with Web Application Firewall
* Configure Web ACL attached to Application Load Balancer
* Create rules against:
  * SQL Injection
  * Cross-site Scripting (XSS)
  * Bad Bots
* Recognize importance of **security-by-design**

#### 4. Attend AI event: "AI-Driven Development Life Cycle"

* Update new trends in software development with AI:
  * AI Development Lifecycle
  * AI-powered backend & testing
  * Developer productivity increase with AI
* Network with industry experts
* Understand **AI-driven software engineering** direction

#### 5. Protect data with AWS KMS

* Create and manage KMS Customer-Managed Key (CMK)
* Practice basic data encryption/decryption
* Understand concepts:
  * Key Policy
  * Key Rotation
  * Envelope Encryption
* Recognize importance of encryption in protecting data on cloud
