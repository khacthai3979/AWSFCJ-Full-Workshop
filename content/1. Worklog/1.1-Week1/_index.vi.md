### **Week 1 Achievements**

Trong tuần đầu tiên, các mục tiêu đặt ra đã được hoàn thành đúng kế hoạch, bao gồm:

#### 1. Thiết lập môi trường học tập và tài khoản AWS
- Tạo tài khoản AWS cá nhân phục vụ thực hành
- Kích hoạt bảo mật MFA cho tài khoản Root
- Tạo **Admin Group** và **Admin User**
- Hoàn thành bước xác thực tài khoản AWS Educate/AWS Account
- Làm quen với giao diện AWS Management Console và điều hướng cơ bản

#### 2. Nắm được kiến thức nền tảng về quản lý chi phí và bảo mật truy cập
- Tìm hiểu cơ chế quản lý chi phí bằng **AWS Budgets**
- Cấu hình cảnh báo chi phí cơ bản
- Tìm hiểu dịch vụ **IAM**, hiểu khái niệm:
  - Users, Groups, Roles
  - Policy & Permission
  - Least-Privilege Access
- Thực hành phân quyền truy cập cho người dùng

#### 3. Triển khai và quản lý mạng ảo với Amazon VPC
- Hiểu được kiến trúc mạng cơ bản trong AWS
- Tạo và cấu hình **VPC**, Subnets (Public/Private), Route Table, Internet Gateway, NAT Gateway
- Kiểm tra kết nối mạng giữa các subnet
- Nắm cách bảo vệ tài nguyên qua **Security Group** và **Network ACL**

#### 4. Áp dụng mô hình Infrastructure as Code
- Thực hành xuất tài nguyên VPC dưới dạng template
- Hiểu được khái niệm **Infrastructure as Code (IaC)**
- Biết cách quản lý hạ tầng bằng code thay vì cấu hình thủ công
- Chuẩn bị cho giai đoạn sử dụng Terraform hoặc CloudFormation

#### 5. Triển khai Amazon EC2 thực tế
- Khởi tạo EC2 Instance đầu tiên
- Cấu hình key pair, security group phù hợp
- Kết nối SSH vào EC2 Instance
- Tìm hiểu khái niệm AMI, Instance Type, Pricing Model
- Dừng/Start/Terminate Instance và cách tiết kiệm chi phí
