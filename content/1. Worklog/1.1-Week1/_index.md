# **WEEK 1 WORKLOG**

### **Week 1 Objectives**

* Get to know new team members and mentors.
* Learn and practice creating a new AWS account, managing AWS spending with AWS Budgets.
* Understand and implement access control management with AWS Identity and Access Management (AWS IAM) and deploy network infrastructure with Amazon Virtual Private Cloud (Amazon VPC).
* Deploy applications on Amazon EC2 and grant applications access to AWS services through IAM Roles.
* Learn additional CCNA basic courses to better understand networking.

### **Week 1 Achievements**

In the first week, the set objectives were completed as planned, including:

#### 1. Set up learning environment and AWS account

* Create personal AWS account for practice
* Activate MFA security for Root account
* Create **Admin Group** and **Admin User**
* Complete AWS Educate/AWS Account verification
* Get familiar with AWS Management Console interface and basic navigation

#### 2. Gain fundamental knowledge about cost management and access security

* Understand cost management mechanism with **AWS Budgets**
* Configure basic cost alerts
* Learn about **IAM** service, understand concepts:
  * Users, Groups, Roles
  * Policy & Permission
  * Least-Privilege Access
* Practice granting access permissions to users

#### 3. Deploy and manage virtual network with Amazon VPC

* Understand basic network architecture in AWS
* Create and configure **VPC**, Subnets (Public/Private), Route Table, Internet Gateway, NAT Gateway
* Check network connectivity between subnets
* Understand how to protect resources through **Security Group** and **Network ACL**

#### 4. Apply Infrastructure as Code model

* Practice exporting VPC resources as templates
* Understand **Infrastructure as Code (IaC)** concept
* Know how to manage infrastructure through code instead of manual configuration
* Prepare for using Terraform or CloudFormation

#### 5. Deploy Amazon EC2 in practice

* Launch first EC2 Instance
* Configure key pair and appropriate security groups
* SSH connect to EC2 Instance
* Understand concepts of AMI, Instance Type, Pricing Model
* Learn how to stop/start/terminate Instance and save costs

---

**Summary:**
Week 1 helps solidify foundational knowledge about **AWS Account Management, IAM, VPC Networking, IaC**, and **EC2**, creating an important technical foundation for the following weeks.
