# **AWS AI/ML Services Overview (Expanded)**

Beyond SageMaker, AWS provides a broad set of managed AI/ML services that help organizations adopt AI fast without building models from scratch.

## **Amazon Rekognition – Computer Vision**

**Capabilities:**

- Object and scene detection  
- Face recognition and emotion analysis  
- Face matching  
- Unsafe content detection  
- Basic text-in-image OCR  

**Pricing:**  
- **$1.00 / 1,000 images** (label detection)  
- **$0.012 / video minute**  
- **$0.001 / face search request**  

---

## **Amazon Comprehend – NLP Service**

**Capabilities:**

- Sentiment analysis  
- Named Entity Recognition (NER)  
- Text classification  
- Key phrase extraction  
- Custom text classifier  

**Pricing:**  
- **$0.0001 / 100 characters**  

---

## **Amazon Transcribe – Speech-to-Text**

**Capabilities:**

- Audio-to-text transcription  
- Speaker identification  
- Custom vocabulary  
- Call analytics  

**Pricing:**  
- **$0.024 / minute** (standard STT)  
- **$0.048 / minute** (analytics)  

---

## **Amazon Polly – Text-to-Speech**

**Capabilities:**

- Natural voice synthesis  
- Wide range of Neural TTS voices  
- Multi-language support  

**Pricing:**  
- **$16 / 1 million characters** (standard)  
- **$20 / 1 million characters** (neural)  

---

## **Amazon Textract – Advanced Document AI / OCR**

**Capabilities:**

- Document OCR  
- Table and form extraction  
- Key-value pair extraction  
- ID analyzer  

**Pricing:**  
- **$1.50 / 1,000 pages** (OCR)  
- **$50 / 1,000 pages** (Forms)  
- **$15 / 1,000 pages** (Tables)  

---

## **Amazon Kendra – Enterprise Search**

**Capabilities:**

- Semantic search  
- Integration with multiple data sources  
- FAQ matching  
- Chatbot integration  

**Pricing:**  
- **$1.50 / hour** (Developer Edition)  
- **$7 / hour** (Enterprise Edition)  

---

## **Amazon Lex – NLP Chatbot**

**Capabilities:**

- Build chatbots for text & speech  
- Intent classification  
- Integrates with Lambda and Contact Center  

**Pricing:**  
- **$0.0075 / text request**  
- **$0.02 / voice request**  

---

## **Amazon Forecast**

- Time-series forecasting  
- Demand forecasting  
- Capacity planning  
- Pay for ingested data and training hours  

---

## **Amazon Personalize**

- Personalized product recommendations  
- Personalized ranking  
- User-item recommendations  

**Pricing:**  
- Pay by:  
  - Data ingestion  
  - Training hours
